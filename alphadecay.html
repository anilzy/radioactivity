Alpha Decay (α‑decay)

What is alpha decay?

Alpha decay is a type of radioactive decay in which an unstable atomic nucleus emits an alpha particle. An alpha particle is identical to the nucleus of a helium‑4 atom — it contains 2 protons and 2 neutrons.

As a result of the emission:

  • The atomic number (number of protons) of the original nucleus decreases by 2.
  • The mass number (total number of protons + neutrons) decreases by 4.

General nuclear equation:

AZX → A‑4Z‑2Y + 42He

For example:

⁺²³⁸U → ²³⁴Th + ⁴He

The emission of the alpha particle allows the original heavy nucleus to move toward a more stable configuration by shedding excess mass and protons.

Why alpha decay occurs

Very heavy nuclei (with many protons and neutrons) tend to be unstable because the repulsive electromagnetic force between protons becomes strong compared to the nuclear binding force. By ejecting an alpha particle, the nucleus reduces both the number of protons and neutrons — which helps it become more stable.
